# godl

Deep Learning library for GoLang

# Supported Operationts

## Tensor Operations

- Add
- Sub
- Mul
- Div
- Pow
- Exp
- Log
- GeMM
- Conv2d
- Conv1d
- Mean

## Activation Functions

- Sigmoid
- Tanh
- ReLU
- Leaky ReLU
- Softmax
- LogSoftmax
- SILU
- GELU

## Other Operations

- Dropout (1d, 2d)
- MaxPool (1d, 2d)
